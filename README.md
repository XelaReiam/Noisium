# Noisium

Browser-based applause meter for conference DemoJam events.

Live: https://xelareiam.github.io/Noisium/

## Stack

React 19, Vite 8, TypeScript, Tailwind v4, Zustand 5

## Local dev

```
npm install
npm run dev
```

## Build

```
npm run build
```

## Status

Pre-v1 / under active development for the 2026-05-08 conference.
